AIM: To create a user friendly tool for educating elderly people in rural areas.
EDU_ROBO is an interactive robot who can be a teacher, a guide as well as a helper.
It can perform various functions: 
    Smart and effective agricultural techniques.
    Information about the precaution and cure of various diseases(e.g. COVID-19).
    Teaches how to use new technologies.
    Government Policies/schemes to benefit people.
    Basic Knowledge about disaster management.
